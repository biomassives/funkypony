Magic URL sign-in (replaces Supabase OTP)

import { Client, Account } from 'appwrite'

const client = new Client()
  .setEndpoint(import.meta.env.VITE_APPWRITE_ENDPOINT)
  .setProject(import.meta.env.VITE_APPWRITE_PROJECT_ID)

const account = new Account(client)

// Send the magic link
await account.createMagicURLToken(
  'unique()',
  email,
  `${window.location.origin}/`
)

// On return — complete session using URL params
const params = new URLSearchParams(window.location.search)
await account.updateMagicURLSession(
  params.get('userId'),
  params.get('secret')
)

After session creation, check localStorage.pendingInvite and call claim-invite as normal.

Appwrite Functions run Node.js (not Deno). The existing Deno edge functions need a port — the logic is identical, only the runtime wrappers change.

  • ✓
    Log in to Appwrite CLI — appwrite login
  • ✓
    Init functions directory — appwrite init function
  • ✓
    mts — email via Mailgun (HTTP trigger)
  • ✓
    claim-invite — invite code redemption + role assignment (HTTP trigger)
  • ✓
    daily-digest — morning summary email (CRON 0 7 * * *)
  • ✓
    mailgun-webhook — bounce tracking (HTTP trigger, set as Mailgun webhook URL)
  • ✓
    daily-call — Twilio outbound voice call (CRON or HTTP trigger)
  • ✓
    twiml-gather — IVR response handler (HTTP trigger)

Node.js function wrapper (mts example)

// functions/mts/src/main.js
import { Client } from 'node-appwrite'
import fetch from 'node-fetch'

export default async ({ req, res }) => {
  const { to, subject, html } = JSON.parse(req.body)
  const form = new URLSearchParams({
    from: `Pantry `,
    to, subject, html
  })
  const r = await fetch(
    `https://api.mailgun.net/v3/${process.env.MAILGUN_DOMAIN}/messages`,
    {
      method: 'POST',
      headers: {
        Authorization: 'Basic ' +
          Buffer.from(`api:${process.env.MAILGUN_API_KEY}`).toString('base64')
      },
      body: form
    }
  )
  return res.json({ ok: r.ok })
}

Environment Variables
Pending
▶

Replace the Supabase vars in your .env.local with these Appwrite equivalents:

# .env.local — Appwrite deployment

# Remove:
# VITE_SUPABASE_URL=...
# VITE_SUPABASE_ANON_KEY=...

# Add:
VITE_APPWRITE_ENDPOINT=https://cloud.appwrite.io/v1
VITE_APPWRITE_PROJECT_ID=your_project_id_here

# Keep (used by functions):
MAILGUN_API_KEY=your_mailgun_key
MAILGUN_DOMAIN=your_mailgun_domain
TWILIO_ACCOUNT_SID=ACxxxx
TWILIO_AUTH_TOKEN=xxxx
TWILIO_FROM_NUMBER=+1xxxxxxxxxx

For self-hosted Appwrite set VITE_APPWRITE_ENDPOINT to https://appwrite.example.org/v1.

Set function-level secrets in Console → Functions → function name → Settings → Variables.

  • ✓
    VITE_APPWRITE_ENDPOINT and VITE_APPWRITE_PROJECT_ID set in .env.local
  • ✓
    Mailgun and Twilio secrets set in each function's variables panel
