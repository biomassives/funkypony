Platform Vision

Where This Is Going

Funky Pony started as one pantry's coordination tool. The long-term goal is a maintained open core with a pluggable component ecosystem — security patches centrally delivered, community features independently grown.

1

Stable Open Core

The GPL-licensed base — pickup queue, logistics hub, volunteer management, calendar, community board, and the E8 ZK integrity layer — receives long-term security maintenance. Pantries deploying it can take security updates without touching their customisations.

2

Pluggable Quasar / Vue Component Architecture

A defined plugin interface for Quasar + Vue 3 components means pantries can extend the UI without forking the core. Dashboard widgets, custom report panels, themed homepage layouts, and role-specific views all become independently installable packages.

3

Community Theme & Feature Marketplace

Designers and developers can publish themes and feature packs — free or paid — and pantry admins install them through the admin panel. Revenue goes to the creator. The platform takes no cut on community contributions; a small platform fee applies to commercial listings to sustain core development.

4

Multi-Pantry Coordination

Regional networks of pantries sharing inventory signals, driver pools, and donor relationships — while each pantry retains full sovereignty over its own data and member list.

5

Security-First Updates

The E8 ZK Lattice integrity layer is actively developed against emerging lattice-based cryptography standards. FIPS-adjacent compliance, tamper-evident audit logs, and post-quantum readiness are first-class goals, not afterthoughts.

Get Involved

Contribute

GPL-3.0 licensed. All contributions welcome. The project is built in public — open issues, open PRs, open design discussions.

💻 Code

Vue 3, Quasar 2, TypeScript, Supabase edge functions. Check the open issues on GitHub — good-first-issue tags are maintained for newcomers.

🎨 Design & Themes

The component library uses CSS custom properties throughout. Build a theme, share it, or list it in the coming marketplace. Light and dark base themes are in the repo.

🌎 Translations

Currently EN, ES, SW (Kiswahili). Every string is in src/i18n/. Adding a language is a single file addition plus a PR.

📖 Documentation

Plain markdown in docs/. If you deployed it, wrote a guide, or found a gap — a PR improving the docs is always welcome.

🎯 Deploy & Report

Run it for a real pantry and file issues. Real-world feedback from food pantry coordinators directly shapes the roadmap. Your pantry's needs are the spec.

💰 Marketplace Listings

When the plugin API ships, list your component, feature pack, or theme. Free community listings always available. Paid listings supported with a small platform fee.
